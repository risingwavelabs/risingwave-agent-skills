# Performance Tuning

Streaming SQL performance edge cases and anti-patterns for RisingWave materialized views. Supplements the `risingwave-troubleshooting` skill with unique findings from production deployments.

## What's Included

4 reference guides across 3 categories.

### Performance Patterns (CRITICAL)

| Reference | Impact | Description |
|-----------|--------|-------------|
| `perf-query-design` | CRITICAL | Window function, aggregation, and UNION edge cases |
| `perf-mv-architecture` | CRITICAL | Dedup placement, distribution skew, backfill strategies |

### Diagnostics (HIGH)

| Reference | Impact | Description |
|-----------|--------|-------------|
| `diag-performance-workflow` | HIGH | Systematic EXPLAIN analysis workflow with red flags and operator reference |

### Review (HIGH)

| Reference | Impact | Description |
|-----------|--------|-------------|
| `review-mv-checklist` | HIGH | Pre-deployment MV performance review checklist |
